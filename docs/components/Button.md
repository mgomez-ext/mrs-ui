## Button

Primary action component with `contained`, `outlined`, and `text` variants; sizes `small`, `medium`, `large`; colors `primary`, `secondary`, `error`, `inherit`.

### Usage

```tsx
import { Button } from '@mrs/ui-v6';

<Button variant="contained" color="primary">Save</Button>;
```

### Tokens

- Color + states: `theme.palette.primary.*`
- Radius: `theme.shape.rounded`
- Spacing: `theme.spacing`

See Storybook for all variants and prop tables.
