## Switch

Binary toggle supporting checked/unchecked, disabled, and focus-visible states.

### Usage

```tsx
import { Switch } from '@mrs/ui-v6';

<Switch defaultChecked color="primary" />;
```

### Tokens

- Thumb/track: `theme.palette.primary.*` and `_states.*`
- Focus ring: `theme.palette.primary._states.focus`
- Radius: `theme.shape.rounded`

See Storybook for full prop tables and label examples.
