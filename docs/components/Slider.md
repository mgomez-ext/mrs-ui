## Slider

Range selection control supporting single-value and range modes with marks and labels.

### Usage

```tsx
import { Slider } from '@mrs/ui-v6';

<Slider defaultValue={30} min={0} max={100} step={5} />;
```

### Tokens

- Track/thumb: `theme.palette.primary.*`
- Radius: `theme.shape.md`
- Spacing: `theme.spacing`

See Storybook for full prop tables and range examples.
